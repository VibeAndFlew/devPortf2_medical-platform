## Workflow Configuration

Each workflow supports:

- **Configurable steps** вҖ” Enable/disable based on organizational policy
- **Role-based step access** вҖ” Different roles see different workflow steps
- **Timeout handling** вҖ” Incomplete workflows flag after configured period
- **Override procedures** вҖ” Emergency bypass with audit notification
- **Template support** вҖ” Saved workflow configurations by specialty/clinic
